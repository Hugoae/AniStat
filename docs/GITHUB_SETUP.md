| Fichier | Usage |

## À faire une fois sur GitHub.com

### About (⚙️ en haut à droite du repo)

- **Description** : `Dashboard de statistiques pour profils AniList publics — graphiques, heatmap, Wrapped`
- **Website** : URL de ton instance déployée
- **Topics** : `anilist`, `anime`, `manga`, `statistics`, `dashboard`, `react`, `typescript`, `vite`, `data-visualization`

### Social preview

1. Settings → General → **Social preview**
2. Upload `docs/social-preview.png`

### Release v1.0.0

1. Releases → **Draft a new release**
2. Tag : `v1.0.0`
3. Titre : `AniStat 1.0 — Lancement public`
4. Copier le contenu de [`CHANGELOG.md`](../CHANGELOG.md) section 1.0.0

### Discussions (optionnel)

Settings → General → Features → cocher **Discussions** pour les questions ouvertes.

### Épingler le repo

Sur ton profil GitHub : **Customize your pins** → ajouter AniStat.
